# cutter

cutter is a CLI for butlerd, the butler daemon.

### Features

  * Auto-completion for requests and notifications (`r YYY<TAB>`)
  * Inline documentation (via `doc`)
  * All the good readline stuff (history via `<C-r>` etc.)

### Usage

```bash
cutter
```
